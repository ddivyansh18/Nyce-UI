Import from CDN

Just add
@import url("https://nyce-ui.vercel.app/nyce.css");
as the first line in your css file, and that's it !

Material Icons

Nyce-UI utilizes Material Icons in many of its components, these components can be imported using a link CSS tag.

Nyce-UI is an open source initiative. Contribute now !
