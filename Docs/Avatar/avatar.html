Avatar

Avatar are images which can be used as user profile pictures, icons, brand logos etc.

To style as an image as avatar, we need to add class "avatar" to the image.

Sizes

Avatars can be of different sizes, this is achieved using "small-avatar", "medium-avatar" and "large-avatar" classes.
